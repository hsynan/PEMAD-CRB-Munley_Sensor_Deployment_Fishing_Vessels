## Data
There are 4 data files available here, one for each of the sensors. <br>
* **SBE_winch.csv** - Seabird 19+ CTD data from the June 2024, June 2025, and August 2025 fishing trips. The seabird 19+ sensor was deployed from a winch during all 3 trips. <br>
* **RBR_headrope.csv.gz** - RBR Brevio CTD data from the June 2024, June 2025, and August 2025 fishing trips. Note that the RBR was deployed from the headrope for the June 2025 and June 2025 trips, but was deployed from a winch during the August 2025 fishing trip. All data is in one file for ease of use, but there are observations from two deployment methods in the file. Due to file size, this file is a .gz, but opens using pandas the same was a csv does. 
* **TDO1_doors.csv** - Lowell TDO-1 (temperature/depth sensor) for the June 2025 fishing trip.
* **TDO1_winch.csv** - Lowell TDO-1 (temperature/depth sensor) for the June 2025 and August 2025 fishing trips.
